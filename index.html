Traversal Order

Breadth-First Search (BFS)

BFS explores all neighbors at the current depth before moving to nodes at the next depth level. It uses a queue data structure and guarantees the shortest path in unweighted graphs. Time Complexity: O(V + E) | Space Complexity: O(V)
